# Spotify Web Player UI Clone

This is a high-fidelity visual clone of the Spotify Web Player built with HTML, CSS, and vanilla JavaScript. 

## Features
- **Responsive Layout**: Replicates the grid layout, fixed sidebar, main content area, and playback bar.
- **Premium Design**: Dark theme, modern aesthetics, smooth gradients, and glassmorphism elements.
- **Interactive Elements**: Play/pause toggle, heart button, interactive progress bars.
- **Dynamic Content**: Time-based greeting ("Good morning", "Good evening").

## Getting Started

To view this project locally, simply open the `index.html` file in your web browser. 

Alternatively, if you have Node.js installed, you can use `serve`:
```bash
npx serve .
```

## How to get a Live Demo Link (Deployment)

To host this clone online for free and get a live demo link:

1. Create a new repository on [GitHub](https://github.com/).
2. Upload these files (`index.html`, `styles.css`, `script.js`, `README.md`) to your repository.
3. Go to the repository **Settings** > **Pages**.
4. Under "Build and deployment", select the `main` branch and click **Save**.
5. GitHub will provide you with a live demo link (e.g., `https://yourusername.github.io/your-repo-name/`).
